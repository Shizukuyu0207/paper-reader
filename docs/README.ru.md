## ✨ Обзор

Skill для [Hermes Agent](https://github.com/henvic/hermes), который читает, анализирует и архивирует научные статьи с 3-уровневым получением контента, интеллектуальным определением предметной области и интеграцией с Obsidian.

## 🎯 Ключевые возможности

| Возможность | Описание |
|-------------|----------|
| 🧠 **Автоопределение области** | 5 областей: молекулярная динамика, медицина, ИИ/ML, биоинформатика, программирование |
| 📊 **3 режима чтения** | Быстрый просмотр (3 мин) · Глубокое чтение · Вопрос-ответ |
| ⚡ **3-уровневое получение контента** | Jina Reader → Scrapling скрытый браузер → web_search fallback |
| 📝 **Архив Obsidian** | YAML frontmatter + структурированные Markdown-заметки |
| 🔍 **Визуальный анализ** | Описание ключевых фигур с помощью ИИ |
| 📦 **Пакетная обработка** | Параллельное получение + параллельный анализ |

## 🔗 Получение контента — Стратегия 3 уровней

Никакой магии — только многоуровневые fallback с честными компромиссами.

| Уровень | Инструмент | Скорость | Вывод | Для |
|---------|-----------|----------|-------|-----|
| **1** | [Jina Reader](https://github.com/jina-ai/reader) | 1-2с | Markdown | arXiv, bioRxiv, открытый доступ, большинство Nature |
| **2** | [Scrapling](https://github.com/D4Vinci/Scrapling) + Camoufox | 5-15с | HTML текст | Уровень 1 не справился, частичный контент Nature/Elsevier |
| **3** | web_search | 2-5с | Только метаданные | Жёсткие пейволы (Cell, NEJM, Lancet) |

### Производительность (реальные данные, май 2026)

| Источник | Уровень | Результат | Время | Контент |
|----------|---------|-----------|-------|---------|
| arXiv PDF | Уровень 1 | ✅ Полный текст | 0.8с | Все разделы |
| bioRxiv PDF | Уровень 1 | ✅ Полный текст | 0.8с | Все разделы |
| Nature Biotechnology | Уровень 1 | ✅ Полный текст | 1.0с | 117K символов |
| Nature Machine Intelligence | Уровень 1→2 | ✅ Полный текст | 6.3с | Jina частично → Scrapling полностью |
| Elsevier/ScienceDirect | Уровень 3 | ⚠️ Метаданные | 3с | Только аннотация + цитирования |

## ⚠️ Честные ограничения

Этот инструмент полезен, но **не всесилен**.

### Что НЕЛЬЗЯ преодолеть

| Сценарий | Реальность | Решение |
|----------|-----------|---------|
| **Жёсткие пейволы** (Cell, NEJM, JAMA) | Требуется институциональный вход или личная подписка | Используйте VPN университета, скачайте PDF вручную и предоставьте как локальный файл |
| **Институциональная аутентификация** (SSO, Shibboleth) | Вход в университетский портал вне компетенции Skill | Скачайте PDF вручную и предоставьте Paper Reader |
| **Свежеопубликованные статьи** | Могут потребоваться дни/недели для индексации | Дождитесь препринта или получите через учреждение |
| **Дополнительные материалы** | Обычно размещаются отдельно | Предоставьте дополнительные файлы отдельно |

### Юридические и этические аспекты

- Мы получаем **только то, что публично видно в браузере**. Если для просмотра нужен логин — залогиньтесь сами.
- Статьи открытого доступа: издатели явно разрешают получение.
- Платный контент: мы не пытаемся обойти аутентификацию или взломать пейвол. Откат к публичным метаданным.
- Ограничение скорости: бесплатный тариф Jina Reader 20 RPM. Мы не бомбардируем серверы издателей.

## 🚀 Установка

### Способ 1: Git Clone (Рекомендуется)

```bash
cd ~/.hermes/skills/
git clone https://github.com/Shizukuyu0207/paper-reader.git
```

### Способ 2: Метод «Ленивого Исследователя»

```
Установи этот skill: https://github.com/Shizukuyu0207/paper-reader

Склонируй в ~/.hermes/skills/paper-reader/,
проверь что MinerU установлен, если нет — пропусти и скажи.
Когда закончишь, подтверди установку и расскажи, что умеет.
```

Идите заваривать чай. 🍵

### Зависимости

| Зависимость | Обязательна | Установка |
|-------------|------------|-----------|
| [Hermes Agent](https://github.com/henvic/hermes) | ✅ Да | См. документацию Hermes |
| [MinerU](https://github.com/opendatalab/MinerU) | ✅ Да | `pip install mineru` |
| [Jina Reader](https://github.com/jina-ai/reader) | Встроен | Использует API `r.jina.ai`, установка не нужна |
| [Scrapling](https://github.com/D4Vinci/Scrapling) | Рекомендуется | `pip install scrapling camoufox && python -m camoufox fetch` |
| Obsidian | Опционально | Для архивных заметок |

## ⚙️ Конфигурация

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
