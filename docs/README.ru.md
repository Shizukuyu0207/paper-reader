## ✨ Обзор

Skill для [Hermes Agent](https://github.com/henvic/hermes) для чтения и анализа научных статей. Принимает URL, локальные PDF или пакеты из 10+ статей — автоматическое получение контента, классификация по домену, глубокий анализ и архивация.

## 🏗️ Архитектура

5-этапный конвейер + 3-уровневая стратегия получения контента. Каждый этап модульный.

```
Получение(3 уровня) → Детекция(5 доменов) → Выбор(4 режима) → Анализ → Архив Obsidian
```

### 3 Уровня Получения

| Приоритет | Инструмент | Скорость | Покрытие |
|-----------|-----------|----------|----------|
| Уровень 1 | Jina Reader | 1-2с | arXiv, bioRxiv, большинство Nature |
| Уровень 2 | Scrapling + Camoufox | 5-15с | Nature/Elsevier при неполном Уровне 1 |
| Уровень 3 | web_search | 2-5с | Жёсткие пейволлы (Cell, NEJM) |
| Локальный | MinerU | ~2мин/40с | Локальные PDF с изображениями |

### Домены Анализа

🧬 Молекулярная динамика | 🏥 Медицина | 🤖 AI/ML | 🔬 Биоинформатика | 💻 Программирование

## 📊 Кейс: Пакетная обработка 9 статей (Май 2026)

| # | Статья | Источник | Метод | Время | Качество |
|---|--------|----------|-------|-------|----------|
| 1 | Allosteric Switches | Nature NBT | MinerU | 93с | ★★★★★ Полный текст, 494 строки + 36 рисунков |
| 2 | ConforNets | arXiv | MinerU | 118с | ★★★★★ Полный текст, 646 строк + 75 рисунков |
| 3-8 | 6 статей с пейволлом | Разные | web_search | ~5с каждая | ★★☆☆☆ Только метаданные |
| 9 | lightning-boltz | GitHub | README | ~3с | ★★☆☆☆ Информация о репозитории |

**Итого**: ~6 минут | После интеграции Jina Reader 4/7 статей обновлены с метаданных до полного текста

### Объективная Оценка

✅ Отличный глубокий анализ для доступных PDF | ✅ 9 статей за 6 минут | ⚠️ Статьи с пейволлом без деталей методов | ⚠️ Elsevier остаётся на Уровне 3

## ⚠️ Честные Ограничения

- **Жёсткие пейволлы** (Cell, NEJM, JAMA): Требуют институциональный доступ → скачайте PDF вручную
- **Статьи Уровня 3**: Заметки без подробных методов и количественных результатов
- Получаем только публично доступный контент браузера.
