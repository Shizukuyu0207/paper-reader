## ✨ Обзор

Skill для [Hermes Agent](https://github.com/henvic/hermes), который читает, анализирует и архивирует научные статьи (PDF) с интеллектуальным определением предметной области и структурированной интеграцией с Obsidian.

Будь то быстрый отбор одного препринта arXiv или пакетная обработка 10+ статей из разных издательств — Paper Reader автоматически выполняет извлечение, классификацию, углублённый анализ и архивирование.

## 🎯 Ключевые возможности

| Возможность | Описание |
|-------------|----------|
| 🧠 **Автоопределение области** | 5 областей: молекулярная динамика, медицина, ИИ/ML, биоинформатика, программирование |
| 📊 **3 режима чтения** | Быстрый просмотр (3 мин) · Глубокое чтение (полный анализ) · Вопрос-ответ (интерактивный) |
| ⚡ **Пакетная обработка** | Параллельная загрузка + извлечение MinerU + параллельный анализ для Paper Alerts |
| 📝 **Архив Obsidian** | YAML frontmatter + структурированные Markdown-заметки в вашем хранилище |
| 🔍 **Визуальный анализ** | Описание и анализ ключевых фигур с помощью ИИ |
| 🔓 **Обход пейвола** | Автоматический fallback на web_search для Nature/Elsevier/bioRxiv |

## 📖 Режимы чтения

### 🔍 Быстрый просмотр
Обзор за 3 минуты для решения о целесообразности глубокого чтения. Извлекает название, аннотацию, ключевые выводы и значимость. Архив не создаётся.

### 📖 Глубокое чтение
Полный структурированный анализ с предметно-ориентированными чеклистами. Генерирует исчерпывающую архивную заметку Obsidian с разбором методов, количественными результатами, ограничениями и выводами для исследования.

### 💬 Режим вопрос-ответ
Интерактивная сессия вопросов и ответов. Задавайте вопросы о содержании статьи, фигурах или методологии. Опциональное сохранение лога Q&A.

### 📦 Пакетный режим (Paper Alert)
Одновременная обработка нескольких статей. Поддерживает смешанные источники:
- **PDF с arXiv** → Прямая загрузка + полное извлечение MinerU
- **Платные статьи** (Nature, Elsevier, bioRxiv) → Fallback на метаданные через web_search
- **Репозитории GitHub** → Анализ README

## 🗂️ Чеклисты по предметным областям

| Область | Ключевые точки анализа |
|---------|----------------------|
| 🧬 **Молекулярная динамика** | Силовые поля, параметры симуляции, RMSD/RMSF, методы свободной энергии, анализ траекторий |
| 🏥 **Медицина** | Дизайн исследования, информация о когортах, статистические методы, клинические исходы, hazard ratios |
| 🤖 **ИИ / ML** | Детали архитектуры, обучающие данные, бенчмарки, сравнение SOTA, вычислительные ресурсы |
| 🔬 **Биоинформатика** | Инструменты пайплайна, статистические тесты, сборка генома, дифференциальная экспрессия, анализ обогащения |
| 💻 **Программирование** | Сложность алгоритмов, системный дизайн, детали реализации, бенчмарки производительности |

## 🚀 Установка

### Способ 1: Git Clone (Рекомендуется)

```bash
cd ~/.hermes/skills/
git clone https://github.com/Shizukuyu0207/paper-reader.git
```

### Способ 2: Метод «Ленивого Исследователя»

Не хотите открывать терминал? Просто киньте ссылку на этот репозиторий вашему Hermes Agent и скопируйте:

```
Установи этот skill: https://github.com/Shizukuyu0207/paper-reader

Склонируй его в ~/.hermes/skills/paper-reader/,
проверь что MinerU установлен (which mineru), если нет — пропусти и скажи мне.
Когда закончишь, подтвердите установку и расскажи, что умеет этот skill.
```

Ваш Agent сделает всё сам. Идите заваривать чай. 🍵

### Способ 3: Ручная установка

Скачайте ZIP и распакуйте в `~/.hermes/skills/paper-reader/`.

## 📝 Пример архивного вывода

Заметки сохраняются в `~/obsidian/papers/{domain}/` с полным YAML frontmatter:

```yaml
---
title: "Artificial allosteric protein switches with ML-designed receptors"
authors: ["Zhong Guo", "David Baker"]
year: 2026
journal: "Nature Biotechnology"
doi: "10.1038/s41587-026-03081-9"
domain: "ai"
tags: [paper/ai, allosteric-switch, biosensor, protein-design]
rating: "5"
---
```

Далее структурированные разделы: Основная информация · Исследовательский вопрос · Методы · Ключевые результаты · Ограничения · Значение для исследования · Сеть цитирования

## ⚙️ Конфигурация

| Переменная | По умолчанию | Описание |
|------------|-------------|----------|
| `MINERU` | Путь к бинарнику MinerU | Путь к исполняемому файлу MinerU |
| `WORK_BASE` | `/tmp/paper-reader` | Временный рабочий каталог |
| `ARCHIVE_BASE` | `~/obsidian/papers` | Корень архива Obsidian |
